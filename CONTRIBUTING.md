# Contribuir

## Commits

Usa Conventional Commits:

- `feat:` nueva solucion.
- `fix:` correccion.
- `docs:` evidencias o documentacion.
- `refactor:` mejora interna.
- `test:` pruebas o validaciones.
- `chore:` ajustes menores permitidos.

## Regla principal

Las entregas solo se aceptan dentro de `resoluciones/nombre-apellido/`.
