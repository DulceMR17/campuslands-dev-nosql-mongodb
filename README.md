# Campuslands Dev NoSQL MongoDB

Repositorio publico de ejercicios de MongoDB y bases de datos NoSQL para estudiantes de Campuslands.

## Objetivo

Practicar modelado documental, CRUD, consultas, aggregation, indices, validacion de esquemas y decisiones de diseno NoSQL con ejercicios contextualizados en temas modernos.

## Ramas principales

- `main`: produccion. No recibe entregas estudiantiles.
- `dev`: integracion. Todos los Pull Requests de estudiantes deben apuntar aqui.

## Niveles incluidos

- [Basico](./basico): 30 ejercicios.
- [Medio](./medio): 30 ejercicios.

## Estructura

```text
campuslands-dev-nosql-mongodb/
├── README.md
├── CONTRIBUTING.md
├── FLUJO-DE-ENTREGA.md
├── GUIA-DE-ENTREGA.md
├── basico/
└── medio/
```

## Flujo rapido

```bash
git clone https://github.com/anndreloopez012/campuslands-dev-nosql-mongodb.git
cd campuslands-dev-nosql-mongodb
git switch dev
git pull origin dev
git switch -c alumno/nombre-apellido/ejercicio-01
```

Entrega solo dentro de `resoluciones/nombre-apellido/` y abre PR hacia `dev`.
